概述

本文主要介绍关于USB2XXX的GPIO操作相关函数，使用这些函数可以控制适配器上P0到P15引脚输出高低电平，或者读取这些引脚电平。在调用这些函数前需要调用设备操作相关函数，比如扫描设备，打开设备等，详情请查看《设备操作相关函数说明》。

下面是本文函数可能会用到的一些宏定义：

  1. //定义函数返回错误代码  
  2. #define GPIO_SUCCESS                          (0)   //函数执行成功  
  3. #define GPIO_ERR_NOT_SUPPORT       (-1)  //适配器不支持该函数  
  4. #define GPIO_ERR_USB_WRITE_FAIL  (-2)  //USB写数据失败  
  5. #define GPIO_ERR_USB_READ_FAIL   (-3)  //USB读数据失败  
  6. #define GPIO_ERR_CMD_FAIL              (-4)  //命令执行失败  
  7.   
  8. //上下拉电阻配置  
  9. #define GPIO_PUPD_NOPULL         0x00    //没有上拉下拉  
  10. #define GPIO_PUPD_UP                  0x01    //使能上拉  
  11. #define GPIO_PUPD_DOWN           0x02    //使能下拉  

GPIO_SetInput

描述：

将指定引脚设置为输入模式，设置为输入模式后可以通过调用GPIO_Read函数获取引脚高低电平状态。

原型：

  1. int WINAPI GPIO_SetInput(int DevHandle,unsigned int PinMask, unsigned char PuPd);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice 函数获得。

PinMask  指定需要设置为输入模式的引脚，该值每个bit位对应一个引脚，最低位对应P0。若对应的bit位为1，则该引脚要被设置为输入模式，为0则不影响引脚原来工作状态。

PuPd  配置引脚模式，0-无上拉和下拉电阻，1-使能上拉电阻，2-使能下拉电阻。

返回值：

引脚配置状态，0-配置引脚成功，若小于0则该函数调用失败。

示例： 

  1. #include <stdio.h>  
  2. #include "usb2gpio.h"  
  3.    
  4. int ret;  
  5. //将P0到P7配置为上拉输入模式  
  6. ret = GPIO_SetInput(DeviceHandle,0x00FF,GPIO_PUPD_UP);  
  7. if(ret != GPIO_SUCCESS){  
  8.     printf("配置引脚失败\n");  
  9.     return;  
  10. }  
  11. //将P8到P15配置为下拉输入模式  
  12. ret = GPIO_SetInput(DeviceHandle,0xFF00,GPIO_PUPD_DOWN);  
  13. if(ret != GPIO_SUCCESS){  
  14.     printf("配置引脚失败\n");  
  15.     return;  
  16. }

GPIO_SetOpenDrain

描述：

将指定引脚设置为开漏模式，在该模式下，引脚可以当双向引脚使用。

原型：

  1. int WINAPI GPIO_SetOpenDrain(int DevHandle,unsigned int PinMask, unsigned char PuPd);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

PinMask  指定需要设置为开漏模式的引脚，该值每个bit位对应一个引脚，最低位对应P0，对应的bit位为1，则该引脚要被设置为开漏模式，为0则不影响引脚原来工作状态。

PuPd  配置引脚模式，0-无上拉或者下拉电阻，1-使能上拉电阻，2-使能下拉电阻。

返回值：

引脚配置状态，0-配置引脚成功，若小于0则该函数调用失败。

示例： 

  1. #include <stdio.h>  
  2. #include "usb2gpio.h"  
  3.    
  4. int ret;  
  5. //将P4到P11配置为上拉开漏模式，开漏模式下需要使能内部上拉电阻或者外接上拉电阻，否则无法输出高电平 
  6. ret = GPIO_SetOpenDrain(DeviceHandle,0x0FF0,GPIO_PUPD_UP);  
  7. if(ret != GPIO_SUCCESS){  
  8.     printf("配置引脚失败\n");  
  9.     return;  
  10. }  

GPIO_Write

描述：

当GPIO引脚被配置为输出模式时，设置引脚输出高电平或者低电平。

原型：

int WINAPI GPIO_Write(int DevHandle,unsigned int PinMask, unsigned int PinValue); 

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

PinMask  指定需要输出数据的引脚，该值每个bit位对应一个引脚，最低位对应P0。对应的bit位为1，则该引脚要根据PinValue对应位的值输出对应的电平状态。对应的bit位为0，则不影响该引脚的电平状态。

PinValue  根据PinMask的值控制引脚输出对应的电平状态，该数据每个bit位对应一个引脚，最低bit位对应P0，若对应的bit位为1且PinMask的对应bit位为1则该引脚输出高电平，若对应bit位为0且PinMask的对应位为1则该引脚输出低电平，否则不影响引脚输出的电平状态。

返回值：

控制引脚输出数据的状态，0-设置成功，若小于0则设置失败。

示例： 

  1. #include <stdio.h>  
  2. #include "usb2gpio.h"  
  3.    
  4. int ret;  
  5. //将P0到P7输出高电平，P8到P15输出低电平  
  6. ret = GPIO_Write(DeviceHandle,0xFFFF,0x00FF);  
  7. if(ret != GPIO_SUCCESS){  
  8.     printf("控制引脚失败\n");  
  9.     return;  
  10. }  
  11. //将P0到P7输出低电平，P8到P15输出高电平  
  12. ret = GPIO_Write(DeviceHandle,0xFFFF,0xFF00);  
  13. if(ret != GPIO_SUCCESS){  
  14.     printf("控制引脚失败\n");  
  15.     return;  
  16. }  
  17. //将P0到P7输出高电平，其他引脚保留原来的电平状态  
  18. ret = GPIO_Write(DeviceHandle,0x00FF,0x00FF);  
  19. if(ret != GPIO_SUCCESS){  
  20.     printf("控制引脚失败\n");  
  21.     return;  
  22. }  

GPIO_Read

描述：

当GPIO引脚被配置为输入模式时，通过该函数获取管脚输入的电平状态。

原型：

  1. int WINAPI GPIO_Read(int DevHandle,unsigned int PinMask, unsigned int *pPinValue);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

PinMask  指定需要获取输入数据的引脚，该值每个bit位对应一个引脚，最低位对应P0。对应的bit位为1，则pPinValue会返回引脚的电平状态，否则不返回状态。

pPinValue  获取引脚电平状态的数据指针。

返回值：

获取引脚输入数据的状态，0-设置成功，若小于0则设置失败。

示例： 

  1. #include <stdio.h>  
  2. #include "usb2gpio.h"  
  3.    
  4. int ret;  
  5. unsigned int PinValue;  
  6. //获取P0到P8引脚的电平状态  
  7. ret = GPIO_Read(DeviceHandle,0x00FF,&PinValue);  
  8. if(ret != GPIO_SUCCESS){  
  9.     printf("获取引脚状态失败\n");  
  10.     return;  
  11. }else{  
  12.     for(int i=0;i<8;i++){  
  13.         if((PinValue&(1<<i))!=0x00){  
  14.             printf("P[%d]为高电平\n",i);  
  15.         }else{  
  16.             printf("P[%d]为低电平\n",i);  
  17.         }  
  18.     }  
  19. }
