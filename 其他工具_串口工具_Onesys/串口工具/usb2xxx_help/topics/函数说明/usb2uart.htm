概述

USB2XXX有2路UART总线，2路UART可以同时使用，每路UART都有一个数据发送控制引脚，配置成485模式之后可以自动输出控制信号，该信号可以跟485收发器芯片的数据方向控制引脚连接，如此就可以实现输出485的信号。

  6.   unsigned char Parity;             //奇偶校验，0-No,4-Even,6-Odd  
  7.   unsigned char TEPolarity;      //TE输出控制，0x80-输出TE信号，且低电平有效，0x81-输出TE信号，且高电平有效，0x00不输出TE信号

UART_Init

描述：

初始化配置UART，必须要调用该函数之后才能调用其他数据收发函数。

原型：

  1. int WINAPI UART_Init(int DevHandle, unsigned char Channel, PUART_CONFIG pConfig);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice 函数获得。

Channel  UART通道索引号，取值0或1。

pConfig  UART初始化配置参数，详细取值见该结构体定义说明。

返回值：

函数执行状态，0表示函数执行成功，其他值表示函数执行失败。

示例：

  1. #include <stdlib.h>  
  2. #include <stdio.h>  
  3. #include <string.h>  
  4. #include "usb_device.h"  
  5. #include "usb2uart.h"  
  6.    
  7. //配置UART总线相关参数  
  8. UART_CONFIG UARTConfig;  
  9. unsigned char UARTIndex = 0;  
  10. UARTConfig.BaudRate = 115200;  
  11. UARTConfig.Parity = UART_PARITY_NO;  
  12. UARTConfig.StopBits = UART_STOP_BITS_1;  
  13. UARTConfig.WordLength = UART_WORD_LENGTH_8BIT;  
  14. ret = UART_Init(DevHandle[0],UARTIndex,&UARTConfig);  
  15. if(ret != UART_SUCCESS){  
  16.     printf("Initialize device error!\n");  
  17.     return 0;  
  18. }  

UART_WriteBytes

描述：

UART发送数据，该函数为阻塞函数，也就是说必须要等到所有数据成功发送完毕之后才返回。

原型：

  1. int WINAPI UART_WriteBytes(int DevHandle,unsigned char Channel,unsigned char *pWriteData,int DataSize);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice 函数获得。

Channel  UART通道索引号，取值0或1。

pWriteData  UART发送数据缓冲区首地址，也就是数据发送指针。

DataSize  待发送的数据字节数。

返回值：

函数执行状态，0表示函数执行成功，其他值表示函数执行失败。

示例：

  1. #include <stdlib.h>  
  2. #include <stdio.h>  
  3. #include <string.h>  
  4. #include "usb_device.h"  
  5. #include "usb2uart.h"  
  6.    
  7. uint8_t UARTIndex = 0;  
  8. uint8_t WriteBuffer[1024];  
  9. //UART发送数据  
  10. for(int i=0;i<sizeof(WriteBuffer);i++){  
  11.     WriteBuffer[i] = i;  
  12. }  
  13. ret = UART_WriteBytes(DevHandle[0],UARTIndex,WriteBuffer,sizeof(WriteBuffer));  
  14. if(ret != UART_SUCCESS){  
  15.     printf("UART write data error!\n");  
  16.     return 0;  
  17. }  

UART_WriteBytesAsync

描述：

UART异步模式发送数据，调用该函数后，数据会发送到适配器中，然后判断上一次的数据是否发送完毕，若已经发送完毕则启动DMA传输数据，函数立即返回，否则等待上一次的数据发送完毕之后再启动DMA开始发送数据，使用此函数可以提高UART发送数据带宽。

原型：

  1. int WINAPI UART_WriteBytesAsync(int DevHandle,unsigned char Channel,unsigned char *pWriteData,int DataSize);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice 函数获得。

Channel  UART通道索引号，取值0或1。

pWriteData  UART发送数据缓冲区首地址，也就是数据发送指针。

DataSize  待发送的数据字节数。

返回值：

函数执行状态，0表示函数执行成功，其他值表示函数执行失败。

示例：

  1. #include <stdlib.h>  
  2. #include <stdio.h>  
  3. #include <string.h>  
  4. #include "usb_device.h"  
  5. #include "usb2uart.h"  
  6.   
  7. uint8_t UARTIndex = 0;  
  8. uint8_t WriteBuffer[1024];  
  9. //UART发送数据  
  10. for(int i=0;i<sizeof(WriteBuffer);i++){  
  11.     WriteBuffer[i] = i;  
  12. }  
  13. ret = UART_WriteBytesAsync(DevHandle[0],UARTIndex,WriteBuffer,sizeof(WriteBuffer));  
  14. if(ret != UART_SUCCESS){  
  15.     printf("UART write data error!\n");  
  16.     return 0;  
  17. }

UART_ReadBytes

描述：

获取UART接收数据缓冲区中的数据，调用该函数后会自动清除已经获取到的数据。

原型：

  1. int WINAPI UART_ReadBytes(int DevHandle,unsigned char Channel,unsigned char *pReadData,int TimeOutMs);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice 函数获得。

Channel  UART通道索引号，取值0或1。

pReadData  UART接收数据缓冲区首地址，也就是数据存储指针。

TineOutMs  接收数据超时时间，若在超时时间内没有接收到数据则会一直等待，直到接收到数据或者超时时间到为止。

返回值：

大于等于0表示接收到的数据字节数，小于0表示读取数据出错。

示例：

  1. #include <stdlib.h>  
  2. #include <stdio.h>  
  3. #include <string.h>  
  4. #include "usb_device.h"  
  5. #include "usb2uart.h"  
  6.   
  7. uint8_t ReadBuffer[10*1024];  
  8. uint8_t UARTIndex = 0;  
  9. int ret,ReadDataNum = 0;  
  10. ret = UART_ReadBytes(DevHandle[0],UARTIndex,ReadBuffer,5000);  
  11. if(ret > 0){  
  12.     ReadDataNum += ret;  
  13.     for(int i=0;i<ret;i++){  
  14.         printf("%02X ",ReadBuffer[i]);  
  15.         if(((i+1)%16)==0){  
  16.             printf("\n");  
  17.         }  
  18.     }  
  19.     printf("\nReadDataNum = %d\n",ReadDataNum);  
  20. }else if(ret < 0){  
  21.     printf("ret = %d\n",ret);  
  22. }
