概述

本文介绍USB2XXX设备操作相关函数，有些函数在进行二次开发时是必须要被调用的，因此若要使用USB2XXX的API接口函数进行二次开发，必须了解下这些函数的基本功能和使用方法。

下面是本文函数可能用到的一些宏定义：

  1. //定义设备信息  
  2. typedef struct _DEVICE_INFO  
  3. {  
  4.     char    FirmwareName[32];   //固件名称字符串  
  5.     char    BuildDate[32];           //固件编译时间字符串  
  6.     int     HardwareVersion;       //硬件版本号  
  7.     int     FirmwareVersion;        //固件版本号  
  8.     int     SerialNumber[3];        //适配器序列号  
  9.     int     Functions;                   //适配器当前具备的功能  
  10. }DEVICE_INFO,*PDEVICE_INFO;  
  11.   
  12. //定义电压输出值  
  13. #define POWER_LEVEL_NONE  0   //不输出  
  14. #define POWER_LEVEL_1V8     1   //输出1.8V  
  15. #define POWER_LEVEL_2V5     2   //输出2.5V  
  16. #define POWER_LEVEL_3V3     3   //输出3.3V  
  17. #define POWER_LEVEL_5V0     4   //输出5.0V  

USB_ScanDevice

描述：

扫描当前连接到电脑上的USB2XXX设备，同时通过pDevHandle返回每个设备的设备号，利用该设备号可以分别控制不同的设备。函数返回当前连接到电脑上的USB2XXX设备数量。

原型：

  1. #ifdef OS_ANDROID  
  2. int  WINAPI USB_ScanDevice(int *pDevHandle,int fd);  
  3. #else  
  4. int  WINAPI USB_ScanDevice(int *pDevHandle);  
  5. #endif  

参数：

pDevHandle  返回每个设备唯一不变的设备号指针，以后对设备的任何操作都必须传入该设备号，注意每个设备的设备号都是不同的，所以在对设备进行任何操作之前都必须调用该函数获得其设备号。

fd 该参数是在安卓系统下用JNA调用我们的函数需要传入的参数，fd的获取方式可以参考我们提供的安卓版本二次开发范例程序源码。

返回值：

当前连接到电脑的设备数量，若返回值小于0，则调用该函数出错。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.     
  4. int ret;  
  5. int DevHandle[20];//假设当前最多连接的设备不超过20个  
  6. ret = USB_ScanDevice(DevHandle);  
  7. if(ret <= 0){  
  8.     printf("无设备连接!\n");  
  9.     return;  
  10. }else{  
  11.     printf("当前连接的设备数为：%d\n",ret);  
  12.     printf("每个设备的句柄为：\n");  
  13.     for(int i=0;i<ret;i++){  
  14.         printf("DevHandle[%d] = %d\n",i,DevHandle[i]);  
  15.     }  
  16. }  

USB_OpenDevice

描述：

打开设备，必须调用该函数，否则后续函数调用无法成功。

原型：

  1. bool WINAPI USB_OpenDevice(int DevHandle);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数或得。

返回值：

打开设备状态。0-打开设备失败，1-打开设备成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. int DevHandle;  
  6. int DevNum = USB_ScanDevice(&DevHandle);//确保当前接入的设备只有一个，否则DevHandle需要使用数组  
  7. if(DevNum > 0){  
  8.     state = USB_OpenDevice(DevHandle);  
  9.     if(state){  
  10.         printf("打开设备成功!\n");  
  11.     }else{  
  12.         printf("打开设备失败！\n");  
  13.         return;  
  14.     }  
  15. }  

USB_CloseDevice

描述：

关闭已经打开的设备，在不需要使用设备的时候可以调用该函数关闭设备，比如在程序退出之前可以调用该函数。

原型：

  1. bool WINAPI USB_CloseDevice(int DevHandle);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

返回值：

打开设备状态。0-关闭设备失败，1-关闭设备成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. state = USB_CloseDevice(DevHandle);  
  6. if(state){  
  7.     printf("关闭设备成功!\n");  
  8. }else{  
  9.     printf("关闭设备失败！\n");  
  10.     return;  
  11. }  

DEV_GetDeviceInfo

描述：

获取设备信息，如固件版本号，固件编译时间，硬件版本号，设备唯一序列号。

原型：

  1. bool WINAPI DEV_GetDeviceInfo(int DevHandle,PDEVICE_INFO pDevInfo,char *pFunctionStr);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

pDevInfo  设备信息结构体指针，PDEVICE_INFO的定义如下所示：

  2. //定义设备信息  
  3. typedef struct _DEVICE_INFO  
  4. {  
  5.     char    FirmwareName[32];   //固件名称字符串  
  6.     char    BuildDate[32];           //固件编译时间字符串  
  7.     int     HardwareVersion;       //硬件版本号  
  8.     int     FirmwareVersion;        //固件版本号  
  9.     int     SerialNumber[3];        //适配器序列号  
  10.     int     Functions;                   //适配器当前具备的功能  
  11. }DEVICE_INFO,*PDEVICE_INFO;  

   pFunctionStr  设备功能描述字符串，类似于“USB2IIC,USB2SPI,USB2PWM,USB2ADC,USB2GPIO,USB2CNT,USB2UART”。

返回值：

获取设备信息是否成功，0-获取设备信息失败，1-获取设备信息成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. DEVICE_INFO DevInfo;  
  6. char    FuncStr[256]={0};  
  7. state =    USB_GetDeviceInfo(DevHandle,&DevInfo,FuncStr);  
  8. if(state){  
  9.     printf("Firmware Name:%s\n",DevInfo.FirmwareName);  
  10.     printf("Firmware Build Date:%s\n",DevInfo.BuildDate);  
  11.     printf("Firmware Version:v%d.%d.%d\n",(DevInfo.FirmwareVersion>>24)&0xFF,(DevInfo.FirmwareVersion>>16)&0xFF,DevInfo.FirmwareVersion&0xFFFF);  
  12.     printf("Hardware Version:v%d.%d.%d\n",(DevInfo.HardwareVersion>>24)&0xFF,(DevInfo.HardwareVersion>>16)&0xFF,DevInfo.HardwareVersion&0xFFFF);  
  13.     printf("Firmware Functions:%08X\n",DevInfo.Functions);  
  14.     printf("Firmware  Functions:%s\n",FuncStr);  
  15. }else{  
  16.     printf("获取设备信息失败！\n");  
  17.     return;  
  18. }  

DEV_EraseUserData

描述：

擦出用户数据存储区域的数据，擦出数据后，默认数据全部为0xFF，用户自定义数据存储区可以存储64KByte数据，但是单次写入数据量不能大于20KByte。

原型：

  1. bool WINAPI DEV_EraseUserData(int DevHandle);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

返回值：

擦出数据状态。0-擦出数据失败，1-擦出数据成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. state = USB_EraseUserData(DevHandle);  
  6. if(!state){  
  7.     printf("Erase user data error!\n");  
  8.     return 0;  
  9. }  

DEV_WriteUserData

描述：

向用户区写入用户自定义数据，单次写入数据不能超过20KByte，总的数据不能超过64KByte。

原型：

  1. bool WINAPI DEV_WriteUserData(int DevHandle,int OffsetAddr,unsigned char *pWriteData,int DataLen);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

OffsetAddr  写入数据在用户区的偏移地址，地址从0x0000开始，小于等于0xFFFF。

pWriteData  待写入的数据缓冲区地址。

DataLen  待写入的数据字节数。

返回值：

写入数据状态。0-写入数据失败，1-写入数据成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. unsigned char UserData[512]={0};  
  6. strcpy((char*)UserData,"Hello user define data!");  
  7. state = USB_WriteUserData(DevHandle,0,UserData,sizeof(UserData));  
  8. if(!state){  
  9.     printf("User data write data error!\n");  
  10.     return 0;  
  11. }  

DEV_ReadUserData

描述：

从用户数据区域读取用户写入的自定义数据，单次读取数据最多不能超过20KByte。

原型：

  1. bool WINAPI DEV_ReadUserData(int DevHandle,int OffsetAddr,unsigned char *pReadData,int DataLen);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

OffsetAddr  读取数据在用户区的偏移地址，地址从0x0000开始，小于等于0xFFFF。

pReadData  待保存读取数据的缓冲区地址。

DataLen  待读取的数据字节数。

返回值：

读取数据状态。0-读取数据失败，1-读取数据成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. bool state;  
  5. unsigned char UserData[512]={0};  
  6. //从用户区读取数据  
  7. state = USB_ReadUserData(DevHandle,0,UserData,sizeof(UserData));  
  8. if(!state){  
  9.     printf("User data write data error!\n");  
  10.     return 0;  
  11. }else{  
  12.     printf("USER DATA = %s \n",UserData);  
  13. }  

DEV_SetPowerLevel

描述：

设置接口输出电平电压，注意，该函数只对带接口电压配置的适配器有用。

原型：

  1. bool WINAPI DEV_SetPowerLevel(int DevHandle,char PowerLevel);  

参数：

DevHandle  设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice函数或得。

PowerLevel  接口电压，0-0V，1-1.8V，2-2.5V，3-3.3V，4-5V。

返回值：

设置接口电压状态。0-设置电压失败，1-设置电压成功。

示例：

  1. #include <stdio.h>  
  2. #include "usb_device.h"  
  3.    
  4. //定义电压输出值  
  5. #define POWER_LEVEL_NONE    0    //不输出  
  6. #define POWER_LEVEL_1V8        1    //输出1.8V  
  7. #define POWER_LEVEL_2V5        2    //输出2.5V  
  8. #define POWER_LEVEL_3V3        3    //输出3.3V  
  9. #define POWER_LEVEL_5V0        4    //输出5.0V  
  10.    
  11. bool state;  
  12. //将接口电压设置为1.8V  
  13. state = DEV_SetPowerLevel(DevHandle,POWER_LEVEL_1V8);  
  14. if(!state){  
  15.     printf("Set powerlevel error!\n");  
  16.     return 0;  
  17. }else{  
  18.     printf("Set powerlevel success\n");  
  19. }
