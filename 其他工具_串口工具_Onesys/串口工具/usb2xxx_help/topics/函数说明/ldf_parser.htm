概述

本文主要介绍LIN总线的LDF文件解析相关函数，这些函数是基于LDFDecoder库函数进行再次封装实现，使用更简单，接口函数也更容易被其他语言调用，此外还增加了直接发送帧数据到LIN总线，或者直接执行调度表的功能，更易于实现LIN总线数据收发。

注意，使用这些函数需要连接我们的适配器才能正常使用，否则无法正常解析数据。

函数的使用范例可以参考我们提供的二次开发范例程序源码。

下面是本文相关函数可能会用到的宏定义：

  1. #define LDF_PARSER_OK                   0//没有错误  
  2. #define LDF_PARSER_FILE_OPEN         (-1)//打开文件出错  
  3. #define LDF_PARSER_FILE_FORMAT       (-2)//文件格式错误  
  4. #define LDF_PARSER_DEV_DISCONNECT    (-3)//设备未连接  
  5. #define LDF_PARSER_HANDLE_ERROR      (-4)//LDF Handle错误  
  6. #define LDF_PARSER_GET_INFO_ERROR    (-5)//获取解析后的数据出错  
  7. #define LDF_PARSER_DATA_ERROR        (-6)//数据处理错误  
  8. #define LDF_PARSER_SLAVE_NACK        (-7)//从机未响应数据  

LDF_ParserFile

描述：

输入LDF文件，解析后输出ldf文件解析句柄，后续函数操作需要该句柄，解析ldf文件后，内部调用LIN初始化函数初始化对应的LIN通道。

原型：

  1. long long WINAPI LDF_ParserFile(int DevHandle, int LINIndex, unsigned char isMaster, char* pLDFFileName);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用 USB_ScanDevice 函数获得。

LINIndex 需要进行数据收发的LIN通道号，0-LIN1,1-LIN2,2-LIN3,3-LIN4。

isMaster 将我们设备节点配置为主机还是从机模式，0-从机，1-主机。

pLDFFileName ldf格式的文件名，建议传入完整的文件路径。

返回值：

文件解析成功则返回ldf解析句柄，后续函数需要使用该句柄，若解析文件失败，则返回0。

示例：

  1. int DevHandle[20];  
  2. int ret = USB_ScanDevice(DevHandle);  
  3. if (!USB_OpenDevice(DevHandle[0])) {  
  4.     printf("打开设备失败！\n");  
  5.     return 0;  
  6. }  
  7. printf("DevHandle = %08X\n", DevHandle[0]);  
  8. long long LDFHandle = LDF_ParserFile(DevHandle[0],0,1, "example.ldf");  
  9. if(LDFHandle == NULL)  
  10. {  
  11.     printf("解析LDF文件失败\n");  
  12.     return 0;  
  13. }

LDF_GetFrameSignalName

描述：

通过信号索引获取帧里面的信号名称。

原型：

  1. int WINAPI LDF_GetFrameSignalName(long long LDFHandle, char* pFrameName, int index, char* pSignalName);  

参数：

LDFHandle ldf文件解析句柄，通过LDF_ParserFile函数获取 。

pFrameName 帧名称。

index 信号索引，不能大于总的信号数量。

pSignalName 信号名称输出。

返回值：

函数执行成功返回0，失败返回负数。

示例：

  1. int FrameLen = LDF_GetFrameQuantity(LDFHandle);  
  2. for (int i = 0;i < FrameLen;i++) {  
  3.     char FrameName[64];  
  4.     if (LDF_PARSER_OK == LDF_GetFrameName(LDFHandle, i, FrameName)) {  
  5.         printf("Frame[%d].Name=%s\n", i, FrameName);  
  6.         int SignalNum = LDF_GetFrameSignalQuantity(LDFHandle, FrameName);  
  7.         for (int j = 0;j < SignalNum;j++) {  
  8.             char SignalName[64];  
  9.             if (LDF_PARSER_OK == LDF_GetFrameSignalName(LDFHandle, FrameName, j, SignalName)) {  
  10.                 printf("\tSignal[%d].Name=%s\n", j, SignalName);  
  11.             }  
  12.         }  
  13.     }  
  14. }

LDF_ExeFrameToBus

描述：

将帧数据发送到LIN总线，或者向从机读取数据，内部会根据主机名称和帧发布者自动判断是发送数据还是读取数据，发送数据校验模式是通过协议版本号自动判断的。

若是发送数据给从机，需要先调用LDF_SetSignalValue函数设置该帧里面每个信号的值。

若是向从机读数据，调用该函数后，可以通过调用LDF_GetSignalValue函数获取读到的值。

原型：

  1. int WINAPI LDF_ExeFrameToBus(long long LDFHandle, char* pFrameName);  

参数：

LDFHandle ldf文件解析句柄，通过LDF_ParserFile函数获取 。

pFrameName 帧名称。

返回值：

函数执行成功返回0，失败返回负数。

示例：

LDF_ExeSchToBus

描述：

执行调度表，调用该函数后会根据调度表里面的帧发送数据或者读取数据。帧间隔时间是通过调度表里面的延时函数控制。

若调度表里面包含发送数据给从机的帧，需要先调用LDF_SetSignalValue函数设置该帧里面每个信号的值。

若调度表包含向从机读数据的帧，调用该函数后，可以通过调用LDF_GetSignalValue函数获取读到的值。

原型：

  1. int WINAPI LDF_ExeSchToBus(long long LDFHandle, char* pSchName);  

参数：

LDFHandle ldf文件解析句柄，通过LDF_ParserFile函数获取 。

pSchName 调度表名称。

返回值：

函数执行成功返回0，失败返回负数。

示例：

  1. //主机写操作，发送数据给从机  
  2. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Reg_Set_Voltage", 13.5);  
  3. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Ramp_Time", 3);  
  4. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Cut_Off_Speed", 4);  
  5. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Exc_Limitation", 15.6);  
  6. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Derat_Shift", 2);  
  7. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "MM_Request", 2);  
  8. LDF_SetSignalValue(LDFHandle, "LIN_CONTROL", "Reg_Blind", 1);  
  9. //执行调度表  
  10. LDF_ExeSchToBus(LDFHandle, "Nissan");  
  11. LDF_GetSignalValueStr(LDFHandle, "LIN_STATE", "MM_State", ValueStr);  
  12. printf("LIN_STATE.MM_State=%s\n", ValueStr);  
  13. LDF_GetSignalValueStr(LDFHandle, "LIN_STATE", "Exc_Duty_Cycle", ValueStr);  
  14. printf("LIN_STATE.Exc_Duty_Cycle=%s\n", ValueStr);  
  15. LDF_GetSignalValueStr(LDFHandle, "LIN_STATE", "Exc_Current", ValueStr);  
  16. printf("LIN_STATE.Exc_Current=%s\n", ValueStr);  
  17. LDF_GetSignalValueStr(LDFHandle, "LIN_STATE", "iStARS_Voltage", ValueStr);  
  18. printf("LIN_STATE.iStARS_Voltage=%s\n", ValueStr);
