概述

基于UDS协议实现的CAN UDS操作函数，使用这些函数可以实现CAN UDS的所有功能。注意，在调用这些函数之前，需要先调用扫描设备，打开设备，CAN初始化的函数。

下面是本文函数可能会用到的一些宏定义：

  1. //函数返回值错误定义  
  2. #define CAN_UDS_OK            0  
  3. #define CAN_UDS_TRAN_USB      -98  
  4. #define CAN_UDS_TRAN_CAN      -99  
  5. #define CAN_UDS_TIMEOUT_A     -100  
  6. #define CAN_UDS_TIMEOUT_Bs    -101  
  7. #define CAN_UDS_TIMEOUT_Cr    -102  
  8. #define CAN_UDS_WRONG_SN      -103  
  9. #define CAN_UDS_INVALID_FS    -104  
  10. #define CAN_UDS_UNEXP_PDU     -105  
  11. #define CAN_UDS_WFT_OVRN      -106  
  12. #define CAN_UDS_BUFFER_OVFLW  -107  
  13. #define CAN_UDS_ERROR         -108  
  14.   
  15. //CAN UDS地址定义  
  16. typedef  struct  _CAN_UDS_ADDR  
  17. {  
  18.     unsigned int    ReqID;        //请求报文ID。  
  19.     unsigned int    ResID;        //应答报文ID。  
  20.     unsigned char   Flag;   //bit[0]-帧类型(0-标准帧，1-扩展帧),bit[1]-FDF(0-普通CAN帧，1-CANFD帧),bit[2]-BRS(0-CANFD帧不加速，1-CANFD帧加速)
  21.     unsigned char   AddrFormats;  //0-normal, 1-extended ,2-mixed
  22.     unsigned char   AddrExt;      //当AddrFormats不为normal时，该数据放到CAN数据域第1字节  
  23.     unsigned char   MaxDLC;       //每帧最大数据字节数，一般设置为8
  24. }CAN_UDS_ADDR;  

CAN_UDS_Request

描述：

按照UDS协议发送数据，被测设备收到这些数据后，会发送响应数据。

原型：

int WINAPI CAN_UDS_Request(int DevHandle,unsigned char CANIndex,CAN_UDS_ADDR *pUDSAddr,unsigned char *pReqData,int DataLen);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

CANIndex 0-对应CAN1,1-对应CAN2。

pUDSAddr UDS协议发送数据时需要用到的相关ID。

pReqData UDS协议里面的SID加上SID所携带的数据。

DataLen pReqData里面的有效数据。

返回值：

函数执行状态，为CAN_UDS_OK表示成功，否则为失败。

示例：

  1. uint8_t req_data[]={0x01,0x0C};  
  2. uint8_t res_data[4096]={0};  
  3. CAN_UDS_ADDR UDSAddr;  
  4. UDSAddr.Flag = 0;//使用标准帧  
  5. UDSAddr.AddrFormats = 0;  
  6. UDSAddr.ReqID = 0x7DF;  
  7. UDSAddr.ResID = 0x7E8;  
  8. UDSAddr.MaxDLC = 8;  
  9. ret = CAN_UDS_Request(DevHandle[DEVIndex],CANIndex,UDSAddr,req_data,sizeof(req_data));  
  10. if(ret != CAN_UDS_OK){  
  11.     printf("CAN UDS request failed! %d\r\n",ret);  
  12. }else{  
  13.     printf("Request:");  
  14.     for(int i=0;i<sizeof(req_data);i++){  
  15.         printf(" %02X",req_data[i]);  
  16.     }  
  17.     printf("\r\n");  
  18. }  

CAN_UDS_Response

描述：

按照UDS协议发送数据后，获取设备响应的数据。

原型：

int WINAPI CAN_UDS_Response(int DevHandle,unsigned char CANIndex,CAN_UDS_ADDR *pUDSAddr,unsigned char *pResData,int TimeOutMs);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

CANIndex 0-对应CAN1,1-对应CAN2。

pUDSAddr UDS协议发送数据时需要用到的相关ID。

pResData  存储设备返回的UDS数据，第一个字节为RSID，后面的为RSID所携带的参数。

TimeOutMs 获取数据超时时间。

返回值：

函数执行状态，大于0，表示获取到的响应字节数，否则为失败。

示例：

  1. ret = CAN_UDS_Response(DevHandle[DEVIndex],CANIndex,UDSAddr,res_data,1000);  
  2. if(ret <= CAN_UDS_OK){  
  3.     printf("CAN UDS response failed! %d\r\n",ret);  
  4. }else{  
  5.     printf("Response:");  
  6.     for(int i=0;i<ret;i++){  
  7.         printf(" %02X",res_data[i]);  
  8.     }  
  9.     printf("\r\n");  
  10.     if((res_data[0]==(0x40+req_data[0]))&&(req_data[1]==res_data[1])){  
  11.         printf("转速 = %d Rpm\r\n",((res_data[2]<<8)|res_data[3])/4);  
  12.     }  
  13. }
