概述

基于UDS协议实现的LIN UDS操作函数，使用这些函数可以实现LIN UDS的所有功能。注意，在调用这些函数之前，需要先调用扫描设备，打开设备，LIN初始化的函数。

下面是本文函数可能会用到的一些宏定义：

LIN_UDS_Request

描述：

按照UDS协议发送数据，被测设备收到这些数据后，会发送响应数据。

原型：

int WINAPI LIN_UDS_Request(int DevHandle,unsigned char LINIndex,LIN_UDS_ADDR *pUDSAddr,unsigned char *pReqData,int DataLen);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

CANIndex 0-对应LIN1,1-对应LIN2。

pUDSAddr UDS协议发送数据时需要用到的相关ID。

pReqData UDS协议里面的SID加上SID所携带的数据。

DataLen pReqData里面的有效数据。

返回值：

函数执行状态，为LIN_UDS_OK表示成功，否则为失败。

LIN_UDS_Response

描述：

按照UDS协议发送数据后，获取设备响应的数据。

原型：

int WINAPI LIN_UDS_Response(int DevHandle,unsigned char LINIndex,LIN_UDS_ADDR *pUDSAddr,unsigned char *pResData,int TimeOutMs);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

CANIndex 0-对应LIN1,1-对应LIN2。

pUDSAddr UDS协议发送数据时需要用到的相关ID。

pResData  存储设备返回的UDS数据，第一个字节为RSID，后面的为RSID所携带的参数。

TimeOutMs 获取数据超时时间。

返回值：

函数执行状态，大于0，表示获取到的响应字节数，否则为失败。

示例：

LIN_UDS_GetMsgFromUDSBuffer

描述：

调用LIN_UDS_Request和LIN_UDS_Response函数后，读取LIN总线上收发的LIN数据，在调用LIN_UDS_Request的时候会清除之前缓冲区中的所有消息。

原型：

int WINAPI LIN_UDS_GetMsgFromUDSBuffer(int DevHandle,unsigned char LINIndex,LIN_EX_MSG *pLINMsg,int BufferSize);  

参数：

DevHandle 设备句柄，本质为设备序号的低4字节，可以通过调用USB_ScanDevice函数获得。

CANIndex 0-对应LIN1,1-对应LIN2。

pLINMsg 存储LIN消息的缓冲区指针。

BufferSize  缓冲区大小。

返回值：

函数执行状态，大于0，表示获取到缓冲区帧数，否则为失败。
