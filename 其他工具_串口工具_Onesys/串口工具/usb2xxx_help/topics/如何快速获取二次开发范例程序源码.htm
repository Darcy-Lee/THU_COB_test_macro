前言

本文收集整理所有关于USB2XXX二次开发的工程及源码，简单的应用示例可能没有专门的文档介绍，但是程序会有详细的注释，稍微复杂的示例会有专门的一篇文档进行介绍。为了方便大家下载，所有工程源码包都放到git上下载。

注意：推荐使用git或者在gitee上下载源码，上面的源码一般都是最新版本的！

使用git克隆

使用

git clone git@gitee.com:toomoss/USB2XXX_Example.git

命令即可将源码克隆到本地。

TortoiseGit使用简易教程

1，下载git并安装；

git软件下载链接：https://git-scm.com/
2，下载TortoiseGit并安装；

TortoiseGit下载链接：https://tortoisegit.org/

3，在将要克隆程序的目录点击鼠标右键，弹出下图所示：

选择Git Clone…

4，在弹出的对话框中填入我们程序的git链接，如下图所示：

点击“OK”即开始拷贝代码文件到本地目录，拷贝界面如下所示：

拷贝完毕之后在本地目录下将有一个USB2XXX_Example目录，里面的文件就是我们最新的二次开发示例源码文件，如下图所示：
